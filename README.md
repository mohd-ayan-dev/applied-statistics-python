## 🚀 Key Concepts Covered

* **Normal Distribution (Bell Curve):** Generating, visualizing, and analyzing normally distributed data sets.
* **Data Sampling Techniques:** Implementing random sampling, stratified sampling, and understanding sample vs. population metrics.
* **Hypothesis Testing (Z-Tests):** Calculating Z-scores, p-values, and determining statistical significance for single samples.

## 💻 How to Run Locally

**1. Clone the repository**
~~~bash
git clone https://github.com/mohdayandev/applied-statistics-python.git
cd applied-statistics-python
~~~

**2. Install required dependencies**
Make sure you have the necessary data science libraries installed:
~~~bash
pip install pandas numpy matplotlib scipy
~~~

**3. Execute the scripts**
Navigate to the concept you want to test and run the Python file:
~~~bash
cd normal-distribution
python your_script_name.py
~~~
